# 🛠️ Supabase Storage 용량 제한 늘리기

현재 업로드하려는 파일(~345MB)이 수파베이스의 기본 파일 크기 제한(보통 50MB)을 초과하여 오류가 발생하고 있습니다. 아래 단계를 따라 제한을 늘려주세요.

## 해결 방법

1.  **Supabase Dashboard**에 접속합니다.
2.  왼쪽 메뉴에서 **Storage**를 클릭합니다.
3.  화면 하단의 **Configuration** 또는 **Settings** (톱니바퀴 아이콘)를 클릭합니다.
4.  **Upload and download settings** 섹션을 찾습니다.
5.  **Maximum File Size** 항목의 값을 늘려줍니다.
    - 예: `500MB` 또는 `1GB` (현재 주차장 데이터가 345MB이므로 최소 400MB 이상 필요)
6.  **Save** 버튼을 눌러 설정을 저장합니다.

## 확인
설정 완료 후, 다시 웹 애플리케이션에서 '저장하기'를 시도해 주세요. 
이후에도 문제가 발생하면 파일이 너무 커서 브라우저 메모리 부족(OOM)이나 네트워크 타임아웃이 발생할 수 있으니, 데이터를 필요한 부분만 추출(CSV 분할)하여 업로드하는 것을 권장합니다.
