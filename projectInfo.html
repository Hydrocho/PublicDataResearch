활용 도구의 역할

세 도구는 각각 다른 역할을 맡습니다. 공공데이터는 근거, NotebookLM은 이해, Colab은 분석을 담당합니다.

자료의 근거

공공데이터

사회 현상을 숫자와 표로 확인할 수 있게 해 주는 기본 자료입니다. 인구, 시설, 예산, 위치 데이터 등을 활용합니다.

질문 탐색·코드 설계

NotebookLM

수집한 데이터와 관련 자료를 바탕으로 탐구 질문을 생성하고, 전처리 방향과 Python 코드 초안을 작성하는 데 활용합니다.

데이터 분석

Google Colab

Python을 이용해 데이터 불러오기, 전처리, 병합, 시각화, 지표 계산을 수행합니다.
